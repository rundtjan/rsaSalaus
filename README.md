## Ohjeet  

Kloonaa repo. Siirry juurikansioon, ja ajaa:  
```bash
poetry install
```
Siirry poetry shelliin:  
```bash
poetry shell
```
ja käynnistä sovellus:
```bash
py src/index.py
```
Windows-koneella:
```bash
python src/index.py
```
